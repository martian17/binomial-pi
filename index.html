Calculating π using binomial distribution

π = 1/(2*σ^2*f(μ)^2) where σ is the standard deviation, f(μ) is the probability distribution on the average point

brief description:

This program calculates π using pascal's triangle and binomial distribution.
this works the same as the random walk one except the source of distribution is deterministic.
